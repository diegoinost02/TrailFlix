Sobre nosotros

Somos Diego Ezequiel Inostroza y Juan Manuel Tretta, dos estudiantes de la Tecnicatura Universitaria en Programación en la Universidad Tencológica Nacional. Como proyecto final decidimos crear TrailFlix, una aplicación web de streaming de películas y series.

Sobre el proyecto

Desarrollamos estre proyecto utilizando Angular como frontend y Json Server como fake backend. El principal objetivo del trabajo fue manejar un CRUD de usuarios con Json Server y consumir una API pública para desarrollar una aplicación, por lo que usando la API de The Movie DB desarrollamos esta app web.

Sobre la app

TrailFlix cuenta con una amplia biblioteca de contenido que se actualiza constantemente. Podras buscar peliculas por categorías o por su nombre, podrás visualizar su trailer y agregar cada una a una lista de favoritos.
